# Test Case Register

This document lists all test cases for the following modules:
- `backend/modules/AuditLogic/auditor.py`
- `backend/modules/Build_pdf/mitigation.py`
- `backend/main.py` (FastAPI endpoints and helpers)
- Shared fixtures and helpers (e.g., `simple_df`, `csv_bytes`).

Each test case is given an ID, module/function, type, purpose, input/condition, and expected result.
